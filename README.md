# 股票分析系统 (Stock Watcher)

一个基于React的现代化股票技术分析系统，支持多维度股票筛选、技术指标分析、持仓管理和数据同步。

## 🚀 功能特性

### 📊 核心功能
- **池子筛选**：支持沪深300、中证500、中证1000、科创50等股票池筛选
- **技术分析选股**：基于多种技术指标进行股票筛选
- **自选股监控**：实时监控自选股票的技术指标
- **持仓股管理**：分组管理持仓股票，支持短线、中线、长线策略
- **回归分析**：历史数据回测和策略验证

### 🎯 技术指标
- **均线系统**：MA8、MA20、MA34、MA55等多周期均线
- **技术指标**：MACD、KDJ、RSI、布林带、威廉指标、CCI等
- **量价分析**：量比、成交量分析
- **趋势判断**：金叉死叉、趋势强度分析

### 📱 界面特性
- **响应式设计**：支持桌面和移动端
- **现代化UI**：基于Ant Design的优雅界面
- **折叠菜单**：可折叠的侧边栏导航
- **Tab分组**：类似东方财富的Tab式分组管理

### 💾 数据管理
- **本地缓存**：分组和持仓数据本地存储
- **数据同步**：支持导出/导入功能，多设备数据同步
- **实时更新**：股票价格和技术指标实时更新

## 🛠️ 技术栈

### 前端技术
- **React 18**：现代化前端框架
- **Vite**：快速构建工具
- **Ant Design**：企业级UI组件库
- **Jotai**：轻量级状态管理
- **React Router DOM**：客户端路由

### 数据服务
- **东方财富API**：股票数据源
- **本地存储**：localStorage数据持久化
- **并发控制**：API请求限流和错误处理

### 开发工具
- **pnpm**：快速包管理器
- **ESLint**：代码质量检查
- **热重载**：开发时实时更新

## 📦 安装和运行

### 环境要求
- Node.js >= 16.0.0
- pnpm >= 7.0.0

### 安装依赖
```bash
pnpm install
```

### 启动开发服务器
```bash
pnpm dev
```

### 构建生产版本
```bash
pnpm build
```

## 🎯 使用指南

### 1. 池子筛选
- 选择股票池（沪深300、中证500等）
- 设置筛选条件
- 执行筛选并查看结果

### 2. 技术分析选股
- 选择技术指标组合
- 设置指标参数
- 批量分析股票

### 3. 自选股监控
- 添加自选股票
- 设置监控策略
- 实时查看技术指标

### 4. 持仓股管理
- 创建分组（短线、中线、长线）
- 添加持仓股票
- 查看买卖评级和置信度

### 5. 数据同步
- 导出数据：点击"导出数据"按钮
- 导入数据：点击"导入数据"按钮，粘贴JSON数据
- 多设备同步：通过导出/导入实现数据同步

## 📊 技术指标说明

### 短线策略（15分钟）
- **MA34和MA55**：双均线趋势跟踪
- **金叉买入**：MA34上穿MA55
- **死叉卖出**：MA34下穿MA55

### 中线策略（日线）
- **MA8和MA20**：中期趋势判断
- **量比分析**：缩量买入，放量卖出
- **趋势确认**：价格突破均线确认趋势

### 长线策略
- **趋势跟踪**：长期趋势分析
- **基本面**：结合基本面分析
